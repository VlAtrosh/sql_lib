# sql_lib

Ключевые классы и методы:
Класс Program
Этот класс является точкой входа в программу и реализует главное меню. В нём выполняется подключение к базе данных, обработка команд и управление взаимодействием с пользователем.

Метод Main:Главный метод программы, запускающий цикл интерфейса пользователя. Предоставляет возможность выбора действий из меню:
Просмотр всех книг с указанием авторов.
Добавление новой книги.
Добавление нового автора.
Поиск книг по имени автора.
Обновление информации о книге.
Удаление книги.
Список всех авторов.
Завершение работы программы.
Методы вывода информации:Используется два метода для печати таблиц книг:
PrintBooksTableWithAuthors выводит список книг вместе с именем автора.
PrintBooksTable показывает список книг без привязки к авторам.
Работа с вводом-выводом:Каждый пункт меню обрабатывается в отдельном блоке, в котором запрашивается необходимая информация у пользователя и производится соответствующее действие с базой данных.
Класс BookStoreDatabase
Класс, обеспечивающий работу с базой данных. Здесь выполняются запросы к PostgreSQL и обработаны исключения.

Конструктор класса принимает четыре параметра:
Хост сервера базы данных.
Название базы данных.
Имя пользователя.
Пароль (может быть пустым).
Основные методы:
TestConnection:Тестирует соединение с базой данных. Возвращает true, если соединение установлено успешно.
AddBook:Добавляет новую книгу в базу данных. Запрашивает всю необходимую информацию о книге (название, ISBN, цену, издательство, дату публикации и автора) и сохраняет её в БД.
GetBooksWithAuthors:Получает список всех книг с привязанными к ним авторами.
GetBooksByAuthorId:Ищет книги, написанные конкретным автором, идентифицированным по его уникальному ID.
UpdateBook:Позволяет обновить данные о книге (наименование и цену) по её ID.
DeleteBook:Удаляет книгу из базы данных по указанному ID.
GetAllAuthors:Возвращает список всех авторов, хранящихся в базе данных.
AddAuthor:Добавляет нового автора в базу данных, запрошенные имя, фамилию и биографию.
Классы сущностей
Программа оперирует тремя ключевыми классами, представляющими сущности:

Book:Представляет запись о книге с полями: ID, название, ISBN, цена, издатель и дата публикации.
BookWithAuthor:Наследует класс Book и добавляет поля, относящиеся к автору книги (ID автора, имя и фамилия).
Author:Хранит информацию об авторе (ID, имя, фамилия и биография).
