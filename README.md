Основные возможности приложения:
Управление книгами:
Просмотр всех книг: вывод полного списка книг с указанием авторов.
Добавление новых книг: ввод информации о книге (название, ISBN, цена, издательство, дата публикации), выбор автора из существующего списка или создание нового.
Обновление информации о книге: изменение названия и цены существующей книги.
Удаление книги: возможность удалить книгу вместе с её связями с автором.
Получение книг без авторов: просмотр списка книг, у которых ещё не указан автор, и возможность добавления автора вручную.
Работа с авторами:
Показ всех авторов: получение списка всех авторов с возможностью просмотра подробной информации (ФИО, биография).
Добавление нового автора: ввод имени, фамилии и биографии автора.
Удаление автора: возможность удаления автора, при условии отсутствия связанных с ним книг.
Дополнительная функциональность:
Возможность выбрать нужное действие из интерактивного меню.
Удобное представление результатов запросов через таблицы с разделением полей и строк.
Контроль ввода данных с соответствующей обработкой ошибок и исключений.
Структура классов:
BookStoreDatabase: Класс обеспечивает взаимодействие с базой данных. Реализует следующие операции:

Тестирование соединения с базой данных.
Получение списка книг с авторами.
Добавление, обновление и удаление книг.
Поиск книг по автору.
Отображение книг без авторов и возможность присвоения автора.
Просмотр и управление списком авторов.
Book: Простой класс моделирующий книгу с основными атрибутами: идентификатор, название, ISBN, цена, издательство и дата публикации.

BookWithAuthor: Расширяет класс Book, добавляя информацию об авторе книги — имя и фамилию.

Author: Модель автора, включающая идентификатор, ФИО и биографию.
